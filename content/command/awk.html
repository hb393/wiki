区切り文字をスペースで指定して1カラム目を抜き出す。

awk -F " " '{print $1}'


特定の文字列の前に空白行を入れる

行頭が Name: または Namespace: または containerd: に一致したら、その行を出力。さらに containerd: の場合は空行も追加

awk 'BEGIN {IGNORECASE=1} /^Name:/ || /^Namespace:/ || /^containerd:/ { print $0 if(/^containerd:/) { print "" } }' input.txt


ワンライナー

awk 'BEGIN{IGNORECASE=1} /^Name:/||/^Namespace:/||/^containerd:/{print; if(/^containerd:/) print ""}' input.txt
