awk

テキスト処理に便利なコマンド。

実行例

オプション 意味
-F 区切り文字を指定
-f ファイルを指定
-v プログラムから参照可能な変数を引数で指定

df

ディスク情報を確認できる

実行例

オプション 意味
-h 読みやすい単位で出力
-a ダミーファイルも含めて表示
-l ローカルファイルシステムのみ表示

du

ディスク使用量を確認できる

実行例

オプション 意味
-a ディレクトリ内のファイル容量も表示
-s サブディレクトリ含めた合計サイズを表示
-S サブディレクトリのサイズを含めないで表示
-b 容量をバイト単位で表示

sar

HWに対するアクセスの統計情報を表示できる

実行例

オプション 意味
-A すべての項目を表示
-b ディスクの入出力と転送レート情報を表示する
-c プロセスの生成回数を表示する
-f ファイルを指定する
-n DEV ネットワーク情報を表示
-r メモリとスワップ情報表示
-u CPU情報表示(iostatとのCPUと同じ)
-P id | ALL CPUごとの情報を表示
-R メモリの統計情報を表示
-W スワップ情報表示

vmstat

ディスク、メモリへの読み書き、スワップの読み書きの統計情報を確認できる。オプショは数字で指定

第1引数何秒ごとに実行するか

第2引数何回実行するか

実行例

表示項目 列 説明
procs r 実行待ちプロセス数
b 割り込み不可能なスリープ状態にあるプロセス数
memory swapd スワップサイズ(KB)
free 空きメモリサイズ(KB)
buff バッファに割り当てられているメモリサイズ(KB)
cache キャッシュに割り当てられているメモリサイズ(KB)
swap si ディスクからスワップインされているメモリサイズ(KB/秒)
so ディスクへスワップアウトされているメモリサイズ(KB/秒)

diff

ファイルの差分を比較できる

実行例

オプション 意味
-i 大文字と小文字を無視
-b スペースの数だけ異なる場合は違いを無視
-E タブ展開によるスペースの変更を無視
-w 空白を無視して比較
-q ファイルが違うかどうかだけを出力する

free

メモリ使用情報を表示できる

実行例

オプション 意味
-b バイト単位で表示
-k KB単位で表示
-m MB単位で表示
-t 物理メモリとスワップメモリの合計行も表示
-0 バッファを増減した行を表示しない
-s 繰り返し実行する間隔(秒)

ps

プロセス単位のCPU使用率を表示できる

実行例

オプション 意味
a 端末上のすべてのプロセスを表示する
e 環境変数も表示する
x 制御端末のないプロセスも表示する
u 実行ユーザ名なども表示する
r 実行中のプロセスのみを表示する
l 長いフォーマットで表示する
U ユーザ名 ユーザ名で選択する
-C コマンド名 コマンド名で選択する
-e すべてのプロセスを表示する
-f 引数等も含めた完全なフォーマットで表示する
-l ロング形式で表示する
-p PID PIDで表示する
-t tty 端末名で選択する
-u UID ユーザIDで選択する
